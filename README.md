# VLSM Calculator

- Data Communications and Network (EE 3019) Course Project.
- VLSM Subnetting Calculator App using ReactJS.

# Tutorials
- Clone repo.
  
```js
git clone https://github.com/LTPhat/DataCommunication-Networking-EE3019.git
```

- Install Node packages.

```js
npm install 
```

- Run app.

```js
npm start vlsm 
```
